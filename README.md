Sistema de Gestão Acadêmica – Universidade EAD (Projeto Integrador SENAC)

! Sobre o Projeto

Este projeto foi desenvolvido como parte do Projeto Integrador do curso Técnico em Desenvolvimento de Sistemas – SENAC EAD.  
O objetivo foi aplicar conhecimentos em modelagem e implementação de banco de dados utilizando MySQL, criando uma solução para gestão de uma instituição de ensino a distância fictícia.

! Objetivo do Sistema

O sistema “Universidade EAD” foi projetado para otimizar o gerenciamento de:

* Alunos
* Professores
* Cursos
* Disciplinas
* Materiais didáticos
* Matrículas

A proposta é substituir o uso de planilhas por um banco de dados estruturado, garantindo organização, integridade e melhor gerenciamento das informações.

! Tecnologias Utilizadas

* MySQL
* SQL
* MySQL Workbench (Modelagem EER)

! Etapas do Projeto

* Levantamento e documentação de requisitos
* Modelagem conceitual e criação do diagrama EER
* Implementação do banco de dados
* Criação de tabelas com chaves primárias e estrangeiras
* Inserção e manipulação de dados

! Competências Aplicadas

* Modelagem de banco de dados relacional
* Definição de relacionamentos
* Normalização
* Criação de scripts SQL
* Organização de projeto técnico
